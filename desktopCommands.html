Desktop Application SQL Commands

To use the database

USE MyDatabase;

To get specific car makes, and models from the database

SELECT DISTINCT model make
FROM vehicle
ORDER BY model;

To get specific car models that are a certain make from the database

SELECT DISTINCT model
FROM vehicle
WHEREA (model = "Toyota")
ORDER BY model;

To get specific years a car make and model was produced from the database

SELECT DISTINCT startYear
FROM vehicle
WHERE (make = "Toyota"
AND model = "Camry")
ORDER BY startYear;

Note that numerous other queries will be used and tested for the desktop application. These queries will be added as the desktop application progresses, and as our team is given what queries are needed for proper execution from the desktop team. Additionally, these queries will be added to this site as they are built and the results from running them will be shown!
