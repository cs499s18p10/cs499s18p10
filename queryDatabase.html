MySQL Commands

To use the database

USE MyDatabase;

The following basic queries can be used to ensure the tables / schemas are set up correctly and that the test data has been inserted into them

SELECT * FROM occupant;

SELECT * FROM vehicle;

SELECT * FROM case;

SELECT * FROM user;

SELECT * FROM workGroup;

SELECT * FROM model;

SELECT * FROM userWorkGroup;

SELECT * FROM reference;

To get the names of all individuals involved in accidents who did not wear a seatbelt

SELECT name FROM occupant WHERE seatbeltUsage = 'n';

To get the count of Toyota vehicles involved in accidents

SELECT COUNT(vin) FROM vehicle WHERE model = 'toyota';
