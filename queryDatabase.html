MySQL Commands & Results

To use the database

USE MyDatabase;

The following basic queries can be used to ensure the tables / schemas are set up correctly and that the test data has been inserted into them

SELECT *
FROM OCCUPANT_INFO;

SELECT *
FROM VEHICLE;

SELECT *
FROM CASES;

SELECT *
FROM USERS;

SELECT *
FROM WORKGROUPS;

SELECT *
FROM MODEL_SPECS;

SELECT *
FROM USER_WORKGROUP;

SELECT *
FROM REFERENCE;

SELECT *
FROM ENVIRONMENT;

To get the names of all individuals involved in accidents who did not wear a seatbelt

SELECT Name
FROM OCCUPANT_INFO
WHERE Seatbelt = 'n';

To get the count of Toyota vehicles involved in accidents

SELECT COUNT(VIN)
FROM VEHICLE
WHERE Model = 'toyota';
